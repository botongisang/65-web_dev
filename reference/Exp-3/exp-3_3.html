HTML stands for HyperText Markup Language.

It is used to structure web pages.

CSS

CSS styles web pages.

It controls layout and design.

color: red;
  • Selectors
  • Properties
JavaScript

JavaScript adds interactivity.

It runs in the browser.

alert("Hello");
  • Variables
  • Functions
Flexbox

Flexbox is used for layout.

It arranges items in rows or columns.

display: flex;
  • justify-content
  • align-items
Grid

Grid is a two-dimensional layout system.

It uses rows and columns.

display: grid;
  • grid-template-columns
  • grid-template-rows
